Kidney Stones Classification NotebookOverviewThis Jupyter notebook implements a deep learning model for classifying kidney diseases from medical images. The model is designed to detect four different conditions:&nbsp; &nbsp; Cyst&nbsp; &nbsp; Tumor&nbsp; &nbsp; Stone&nbsp; &nbsp; NormalDataset StructureThe dataset is organized into four main directories, each containing images of the corresponding kidney condition:&nbsp; &nbsp; Cyst: 3,709 images&nbsp; &nbsp; Tumor: 2,283 images&nbsp; &nbsp; Stone: 1,377 images&nbsp; &nbsp; Normal: 5,077 imagesDependenciesThe notebook requires the following Python libraries:Core Libraries&nbsp; &nbsp; numpy - Numerical computations&nbsp; &nbsp; pandas - Data manipulation&nbsp; &nbsp; matplotlib - Data visualization&nbsp; &nbsp; seaborn - Statistical data visualization&nbsp; &nbsp; tensorflow - Deep learning framework&nbsp; &nbsp; keras - High-level neural networks API&nbsp; &nbsp; opencv-python (cv2) - Image processing&nbsp; &nbsp; scikit-learn - Machine learning utilities
